Read an Eigen Matrix to a csv line.

Parameters
lineThe CSV line where the data should be placed.
aThe Eigen Matrix to read.
Returns
A reference to the csv line with the data emplaced.

Eigen matrices are stored into a csv in row-major order. Note that no information about the matrix size is included.
